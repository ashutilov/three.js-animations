# Project Title

Sandbox for three.js


## Getting Started

Install all required npm's

```
npm install
```

To start dev server - go to public dir

```
cd public/
```


## Installing

To create bundle - online update during development

```
npm run watch
```

## Deployment

To create bundle - one time command

```
npm run build
```
